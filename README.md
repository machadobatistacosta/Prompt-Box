**Prompt Box** is a desktop ecosystem designed to help European enterprises leverage Large Language Models (LLMs) without data exfiltration risks. We bring the AI to the data, ensuring 100% GDPR compliance and Data Sovereignty.

---

## 🚀 Key Features

* **🔒 100% Offline Architecture:** No data ever leaves the local machine. Inference runs on-device (Edge AI).
* **⚡ Consumer Hardware Optimized:** Proprietary RAG orchestration layer optimized for Intel i5/i7 CPUs and NPUs. No expensive H100 GPUs required.
* **📚 Local RAG (Retrieval-Augmented Generation):** Ingest PDF/DOCX confidential files into a local Vector Database (ChromaDB) for grounded answers.
* **⚖️ Domain-Specific Fine-Tuning:** Pre-packaged adapters (LoRA) for Legal (Brazilian CLT, EU GDPR) and Finance sectors.

---

## 🛠️ Tech Stack

* **Core:** Python, Streamlit
* **Orchestration:** LangChain / Custom Pipeline
* **Vector Database:** ChromaDB (Local Persistence)
* **Inference Engine:** Ollama / Llama.cpp
* **Supported Models:** Llama 3 (8B), Phi-3 Mini, Mistral 7B
